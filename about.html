About This Project

Welcome to Pokémon Info — a fun and informative website built to help you discover and explore Pokémon using real-time data from the PokéAPI.

This project is a simple frontend web app built using HTML, CSS, and JavaScript. It helps users search for Pokémon by name or number, and view details such as type, stats, abilities, and more.

What You'll Find Here

  • Pokémon listings with name, ID, and image
  • Live data fetched using PokéAPI
  • Clean UI and responsive design
  • Easy navigation and mobile-friendly interface

Technologies Used

  • HTML5 & CSS3
  • Vanilla JavaScript
  • PokéAPI (pokeapi.co)
  • Responsive Web Design

This site is made for learning, portfolio, and for all Pokémon fans. Thanks for visiting!
